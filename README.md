## Project Overview

**BOOKNest** is an Online Book Store Website developed as part of the **TYBSc Computer Science** curriculum. The project demonstrates the practical implementation of web development concepts including frontend design, basic backend integration, and version control using Git and GitHub.

## Academic Details

* **Course:** TYBSc Computer Science
* **College:** KTHM College, Nashik
* **University:** Savitribai Phule Pune University (SPPU)

## Project Description

The BOOKNest website allows users to browse books, read blogs related to books, and add books to a cart. The main focus of the project is to create a user-friendly interface and understand the structure of a real-world web application.

---

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript (basic)
* **Backend:** PHP
* **Tools:** Visual Studio Code, Git, GitHub

---

## Key Features

* Book listing with images
* Blog section for book-related content
* Add to cart functionality
* Reusable header and footer components
* Simple and clean user interface
